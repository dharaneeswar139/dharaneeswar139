### 🚀 Featured Project

<details>
<summary><strong>🌿 Plant Disease Detection using CNN</strong></summary>
<br>

A web-based plant disease detection system built around a CNN trained on the PlantVillage dataset, with a Django backend serving real-time predictions from uploaded leaf images.

| | |
|---|---|
| **Stack** | Python, TensorFlow, Django, CNN, OpenCV |
| **Core Function** | Real-time disease classification from leaf images |
| **Interface** | Responsive UI for image upload, prediction, and treatment recommendations |
| **Optimization** | Tuned for improved prediction accuracy and inference speed |
| **Use Case** | Practical agricultural decision support |

- Trained and integrated a CNN model with a Django backend for real-time inference
- Built an end-to-end pipeline: image upload → prediction → treatment recommendation
- Focused on making the model fast and accurate enough for real-world use

</details>
